# smartparking_client
Autodesk Hackathon smart parking client

# setup guide (Mac)

- install XCode v7.3+

- install [node](https://nodejs.org/en/)
- install react native cli 
`npm install -g react-native-cli`

- do command below
```
cd AusomeParking
npm install
react-native run-ios
```
